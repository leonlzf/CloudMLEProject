# Dataset

Place the ECG5000 dataset files in this directory before running the local pipeline.

Expected files:

- `ECG5000_TRAIN.txt`
- `ECG5000_TEST.txt`
- `ECG5000_TRAIN.arff`
- `ECG5000_TEST.arff`
- `ECG5000_TRAIN.ts`
- `ECG5000_TEST.ts`
- `ECG5000.txt`

The project code uses the `.txt` files by default.
